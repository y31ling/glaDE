格式说明
📄 文件结构
lens <类型> <z> p1 p2 p3 p4 p5 p6 p7 point <z_s> <x_s> <y_s>
· 最少 1 行 lens + 1 行 point 即可运行
· 支持任意 glafic 模型类型，任意数量
· 参数不足 7 个时自动补 0，超出 7 个截断
· # 开头为注释行，空行忽略
· 数值支持科学计数法 (1.2e+09)
🌟 Sérsic sers 星系光度分量
lens sers z M x₀ y₀ Rₑ n e PA
M质量/光度 [M☉]
x₀, y₀中心位置 [arcsec]
Rₑ有效半径 [arcsec]
nSérsic 指数 （椭圆≈4，盘≈1）
e椭率 [0,1)
PA位置角 [deg] 从北向东
⚡ 等温球/椭球 sie sis
lens sie z σ x₀ y₀ e PA 0 0 lens sis z σ x₀ y₀ 0 0 0 0
σ速度弥散 [km/s]
x₀, y₀中心位置 [arcsec]
e椭率 （sis 固定为 0）
PA位置角 [deg]
🌀 NFW 暗物质晕 anfw gnfw
lens anfw z M_vir c x₀ y₀ 0 0 0 lens gnfw z M_vir x₀ y₀ 0 0 c 1.0
M_vir维里质量 [M☉]
c浓度参数 （典型 5–50）
x₀, y₀中心位置 [arcsec]
1.0gnfw 外斜率 γ，固定为 1
💫 Pseudo-Jaffe jaffe
lens jaffe z σ x₀ y₀ a 0 0 rco
σ速度弥散 [km/s]
x₀, y₀中心位置 [arcsec]
a截断半径 [arcsec]，须 > rco
rco核心半径 [arcsec]，≥ 0
👑 King 球状星团 pgc king
lens pgc z M x₀ y₀ 0 0 rc c
M总质量 [M☉]，典型 10⁴–10⁸
x₀, y₀中心位置 [arcsec]
rc核半径 [arcsec]
c集中度 log₁₀(rt/rc) （典型 0.8–2.2）
⚫ 点质量 point pnt
lens point z M x₀ y₀ 0 0 0 0
M质量 [M☉]
x₀, y₀位置 [arcsec]
🔵 其他常用类型
lens tnfw z M_vir x₀ y₀ 0 0 c rt # 截断NFW lens hern z M x₀ y₀ rs 0 e PA # Hernquist
tnfw: rt = 截断半径 [arcsec]
hern: rs = 特征半径 [arcsec]
📍 背景点源 point （恰好 1 行）
point z_s x_s y_s
z_s源红移 （如 0.4090）
x_s源位置 x [arcsec]
y_s源位置 y [arcsec]

最简（1 lens + 1 point）：
lens sie 0.2160 118.3 0.00266 0.02758 0.157 29.2 0.0 0.0 point 0.4090 2.685e-03 2.444e-02
完整（iPTF16geu，2×Sersic + SIE）：

📄 File Structure
lens <type> <z> p1 p2 p3 p4 p5 p6 p7 point <z_s> <x_s> <y_s>
· Minimum: 1 lens line + 1 point line to run
· Supports any glafic model type, any number
· Params fewer than 7 are auto-padded with 0, excess truncated
· Lines starting with # are comments, blank lines ignored
· Values support scientific notation (1.2e+09)

lens sers z M x₀ y₀ Rₑ n e PA
MMass/Luminosity [M☉]
x₀, y₀Center Position [arcsec]

nSérsic Index (elliptical≈4, disk≈1)
eEllipticity [0,1)
