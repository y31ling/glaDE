# iPTF16geu 超新星引力透镜系统 - 双Sersic+SIE模型详解

## 背景

**iPTF16geu** 是一个Ia型超新星，通过前景透镜星系形成了一组爱因斯坦环(Einstein Ring)像。

### 科学问题
- **放大率异常(Magnification Anomalies)**: 早期使用单一椭圆透镜模型虽能很好还原成像位置，但预测的放大率与观测亮度比存在系统性偏差
- **透镜星系形态**: 观测表明这是一个**棒旋星系**(Barred Spiral Galaxy)，具有复杂的非轴对称质量分布

### 解决方案
论文作者提出使用**双Sersic轮廓 + SIE模型**来更准确地描述棒旋星系的质量分布：
- 第一个Sersic轮廓：核球(Bulge)成分
- 第二个Sersic轮廓：盘/棒(Disk/Bar)成分  
- SIE模型：暗物质晕(Dark Matter Halo)

## 文件说明

### 1. bestfit.dat - 最佳拟合参数
包含透镜模型的所有参数，是整个模拟的核心输入文件。

#### 透镜成分1：Sersic轮廓（核球）
```
lens sers 0.2160 9.896617e+09 2.656977e-03 2.758473e-02 2.986760e-01 1.124730e+02 3.939718e-01 1.057760e+00
```
- **红移**: 0.216
- **速度弥散**: 9.9×10⁹ m/s → 对应质量约10¹⁰ M☉
- **中心位置**: (0.0027", 0.0276")
- **椭率**: 0.299 (较圆)
- **位置角**: 112.5°
- **有效半径**: 0.394" 
- **Sersic指数**: 1.06 ≈ 1 (接近指数盘)

#### 透镜成分2：Sersic轮廓（盘/棒）
```
lens sers 0.2160 2.555580e+10 2.656977e-03 2.758473e-02 4.242340e-01 5.396370e+01 1.538855e+00 1.000000e+00
```
- **速度弥散**: 2.6×10¹⁰ m/s → 质量更大，是主导成分
- **椭率**: 0.424 (明显更扁，符合盘/棒的特征)
- **位置角**: 54.0° (与第一个成分相差约60°，说明有棒结构)
- **有效半径**: 1.54"
- **Sersic指数**: 1.0 (标准指数盘)

#### 透镜成分3：SIE模型（暗物质晕）
```
lens sie 0.2160 1.183382e+02 2.656977e-03 2.758473e-02 1.571203e-01 2.920348e+01 0.0 0.0
```
- **速度弥散**: 118 km/s (典型星系暗物质晕)
- **椭率**: 0.157 (相对圆，符合暗物质晕预期)
- **位置角**: 29.2°

#### 点源：超新星
```
point 0.4090 2.685497e-03 2.443616e-02
```
- **源红移**: 0.409 (比透镜红移0.216更远)
- **源位置**: (0.0027", 0.0244") 在源平面

### 2. SN_lens_vSN_2Sersic_SIE_point.dat - 成像结果

超新星被透镜成**4个像**（四重像系统）：

| 像 | 位置 (x, y) | 放大率 | 时间延迟(天) | 类型 |
|---|------------|--------|------------|------|
| A | (-0.104", -0.254") | +15.1 | 0.000 | 最小像(正宇称) |
| B | (+0.281", -0.032") | -30.9 | 0.164 | 鞍点像(负宇称) |
| C | (-0.223", +0.195") | -20.2 | 0.207 | 鞍点像(负宇称) |
| D | (+0.142", +0.288") | +20.7 | 0.094 | 最大像(正宇称) |

**关键观测量**：
- **放大率比**: A:B:C:D = 15:31:20:21
- **时间延迟**: 最大差异0.207天，可用于测量哈勃常数
- **宇称**: 2个正宇称 + 2个负宇称（典型四重像配置）

### 3. SN_lens_vSN_2Sersic_SIE_optresult.dat - 优化过程

展示了参数优化的两次迭代：

**优化指标**：
- **自由参数数**: 13个（两个Sersic各6个参数 + SIE的4个参数 - 3个固定参数）
- **χ²值**: 2.6×10⁻⁶ (极好的拟合)
- **模型评估次数**: 
  - 第一次运行: 1005次
  - 第二次运行: 75次（精调）

**收敛判据**: 两次运行得到相同的χ²和参数值，表明找到了全局最优解

### 4. SN_lens_vSN_2Sersic_SIE_crit.dat - 临界曲线

包含863条线段，描绘临界曲线(Critical Curves)和焦散线(Caustics)的形状。

**物理意义**：
- **临界曲线**: 透镜平面上放大率→∞的位置
- **焦散线**: 临界曲线在源平面的映射
- **成像规则**: 
  - 源在焦散线内 → 4个像（当前情况）
  - 源在焦散线上 → 产生/消失像的临界位置
  - 源在焦散线外 → 2个像

## 物理解释

### 为什么需要双Sersic模型？

1. **棒旋星系的复杂性**：
   - 核球：近似球形，Sersic指数~1-2
   - 棒：扁平，椭率大，与盘有不同的位置角
   - 暗物质晕：弥散分布，椭率小

2. **单一椭圆模型的局限**：
   - 无法同时匹配位置和放大率
   - 棒结构引起的非对称性被忽略
   - 导致系统性放大率偏差

3. **双Sersic的优势**：
   - 两个椭圆成分可以有不同的椭率和位置角
   - 更灵活地描述实际质量分布
   - 减少放大率异常

### 参数的物理含义

- **速度弥散(σ)**: 反映质量 M ∝ σ² R
- **椭率(e)**: e = 1 - q，其中q是轴比
- **位置角(θ)**: 长轴相对于北向的夹角
- **Sersic指数(n)**: 
  - n=1: 指数盘
  - n=4: de Vaucouleurs轮廓(椭圆星系)
  - n>4: 更集中的轮廓

### 时间延迟的应用

不同像的时间延迟可以用来：
1. **测量哈勃常数H₀**: Δt ∝ 1/H₀
2. **约束透镜质量分布**: 时间延迟对质量分布敏感
3. **超新星光变曲线研究**: 可以"预测"后续像的爆发时间

## 与观测的比较

使用双Sersic+SIE模型后：
- ✓ 像位置拟合极好 (χ² ~ 10⁻⁶)
- ✓ 一定程度缓解了放大率异常
- ✓ 物理上更合理（符合棒旋星系形态）

**仍然可能存在的问题**：
- 小尺度结构（暗物质子晕）的影响
- 沿视线方向其他质量的干扰
- 微引力透镜效应（透镜星系中的恒星）

## glafic使用示例

要重现这个模拟，可以使用bestfit.dat作为输入：

```bash
# 计算像位置和放大率
./glafic bestfit.dat findimg

# 计算临界曲线
./glafic bestfit.dat calcimage

# 运行优化（如果要调整参数）
./glafic bestfit.dat optimize
```
